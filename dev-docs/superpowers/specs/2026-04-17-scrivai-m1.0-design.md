## 4. `LLMClient` 内部实现详细设计

### 4.1 数据结构

```python
# scrivai/pes/llm_client.py

@dataclass
class LLMResponse:
    """SDK 调用的统一返回。"""
    result: str                  # ResultMessage.result(Agent 最终文本)
    turns: list[PhaseTurn]       # 解析后的 turn 序列(供 BasePES 写入 PhaseResult)
    usage: dict[str, Any]        # ResultMessage.usage(token 统计)
    duration_ms: int             # ResultMessage.duration_ms
    session_id: str | None       # ResultMessage.session_id


# 模块内部异常,不暴露给业务层
class _MaxTurnsError(Exception):
    def __init__(self, num_turns: int):
        self.num_turns = num_turns
        super().__init__(f"max_turns reached at {num_turns}")


class _SDKExecutionError(Exception):
    def __init__(self, stop_reason: str | None, errors: list[str]):
        self.stop_reason = stop_reason
        self.errors = errors
        super().__init__(f"stop_reason={stop_reason} errors={errors}")
```

### 4.2 `LLMClient.execute_task()` 完整伪代码

```python
class LLMClient:
    def __init__(self, model: ModelConfig):
        self.model = model

    def _build_options(
        self, *,
        system_prompt: str,
        allowed_tools: list[str],
        max_turns: int,
        permission_mode: str,
        cwd: Path,
        extra_env: dict[str, str] | None,
    ) -> ClaudeAgentOptions:
        env: dict[str, str] = dict(extra_env or {})
        if self.model.base_url:
            env["ANTHROPIC_BASE_URL"] = self.model.base_url
        if self.model.api_key:
            env["ANTHROPIC_AUTH_TOKEN"] = self.model.api_key
        return ClaudeAgentOptions(
            model=self.model.model,
            system_prompt=system_prompt,
            allowed_tools=allowed_tools,
            max_turns=max_turns,
            permission_mode=permission_mode,
            cwd=str(cwd),
            env=env,
            mcp_servers={},                # design §5.3.2:不使用 MCP
            setting_sources=["project"],   # 让 SDK 读 workspace/.claude/{agents,skills}/
        )

    async def execute_task(
        self, *,
        prompt: str,
        system_prompt: str,
        allowed_tools: list[str],
        max_turns: int,
        permission_mode: str,
        cwd: Path,
        extra_env: dict[str, str] | None = None,
        on_turn: Callable[[PhaseTurn], None] | None = None,
    ) -> LLMResponse:
        """单次 SDK query。无内部重试,SDK 异常自然冒泡。"""
        options = self._build_options(
            system_prompt=system_prompt,
            allowed_tools=allowed_tools,
            max_turns=max_turns,
            permission_mode=permission_mode,
            cwd=cwd,
            extra_env=extra_env,
        )

        turns: list[PhaseTurn] = []
        pending_tool_calls: dict[str, dict] = {}   # tool_use_id → {name, input, result, is_error}
        result_response: LLMResponse | None = None

        async for message in query(prompt=prompt, options=options):
            if isinstance(message, AssistantMessage):
                turn = self._parse_assistant_turn(message, len(turns), pending_tool_calls)
                if turn:
                    turns.append(turn)
                    if on_turn:
                        on_turn(turn)

            elif isinstance(message, UserMessage):
                turn = self._parse_user_turn(message, len(turns), pending_tool_calls)
                if turn:
                    turns.append(turn)
                    if on_turn:
                        on_turn(turn)

            elif isinstance(message, ResultMessage):
                if message.is_error:
                    if message.stop_reason == "max_turns":
                        raise _MaxTurnsError(message.num_turns)
                    raise _SDKExecutionError(
                        stop_reason=message.stop_reason,
                        errors=message.errors or [],
                    )
                result_response = LLMResponse(
                    result=message.result or "",
                    turns=turns,
                    usage=message.usage or {},
                    duration_ms=message.duration_ms,
                    session_id=message.session_id,
                )

            # SystemMessage / StreamEvent / RateLimitEvent:忽略(MVP,与 Herald2 一致)

        if result_response is None:
            raise RuntimeError("未收到 ResultMessage")
        return result_response
```

### 4.3 消息解析方法(对齐 Herald2 `core/llm.py:23-36 + 137-186`)

```python
def _text_from_content(content: str | list[dict[str, Any]] | None) -> str:
    """从 ToolResultBlock.content 提取纯文本(借 Herald2 helper)。"""
    if content is None:
        return ""
    if isinstance(content, str):
        return content
    parts: list[str] = []
    for item in content:
        if isinstance(item, dict):
            text = item.get("text", "")
            if text:
                parts.append(text)
    return "\n".join(parts)


def _parse_assistant_turn(
    self,
    msg: AssistantMessage,
    turn_index: int,
    pending: dict[str, dict],
) -> PhaseTurn | None:
    """AssistantMessage → 1 个 PhaseTurn。

    - 多个 TextBlock 拼接为 data["text"]
    - 多个 ToolUseBlock 收集到 data["tool_uses"];同时填 pending dict
    - ThinkingBlock 在 MVP 忽略(节省 trajectory 体积)
    """
    text_parts: list[str] = []
    tool_uses: list[dict] = []
    for block in msg.content:
        if isinstance(block, TextBlock):
            text_parts.append(block.text)
        elif isinstance(block, ToolUseBlock):
            tool_uses.append({
                "id": block.id,
                "name": block.name,
                "input": block.input,
            })
            pending[block.id] = {
                "name": block.name,
                "input": block.input,
                "result": None,
                "is_error": None,
            }
        # ThinkingBlock / ToolResultBlock 在 assistant message 中忽略
    if not text_parts and not tool_uses:
        return None
    return PhaseTurn(
        turn_index=turn_index,
        role="assistant",
        content_type="tool_use" if tool_uses else "text",
        data={
            "text": "".join(text_parts),
            "tool_uses": tool_uses,
            "model": msg.model,
            "stop_reason": msg.stop_reason,
            "usage": msg.usage,
        },
        timestamp=datetime.now(timezone.utc),
    )


def _parse_user_turn(
    self,
    msg: UserMessage,
    turn_index: int,
    pending: dict[str, dict],
) -> PhaseTurn | None:
    """UserMessage(tool result)→ 1 个 PhaseTurn。

    - 仅处理 list 形式 content(纯字符串 user message MVP 不构造)
    - 通过 tool_use_id 配对 pending dict,合成 stdout/stderr/exit_code(Herald2 模式)
    - 同时回填 pending[id]["result"](内部状态,不写 turn)
    """
    if not isinstance(msg.content, list):
        return None
    cli_result = (
        msg.tool_use_result
        if isinstance(msg.tool_use_result, dict)
        else {}
    )
    tool_results: list[dict] = []
    for block in msg.content:
        if not isinstance(block, ToolResultBlock):
            continue
        target = pending.get(block.tool_use_id)
        result = {
            "tool_use_id": block.tool_use_id,
            "tool_name": target["name"] if target else None,
            "content": block.content,
            "is_error": block.is_error,
            "stdout": cli_result.get("stdout") or _text_from_content(block.content),
            "stderr": cli_result.get("stderr", ""),
            "exit_code": 1 if block.is_error else 0,
        }
        tool_results.append(result)
        if target:
            target["result"] = block.content
            target["is_error"] = block.is_error
    if not tool_results:
        return None
    return PhaseTurn(
        turn_index=turn_index,
        role="user",
        content_type="tool_result",
        data={"tool_results": tool_results},
        timestamp=datetime.now(timezone.utc),
    )
```

### 5.2 `_call_sdk_query` 实现(替换原 NotImplementedError)

```python
async def _call_sdk_query(
    self,
    phase_cfg: PhaseConfig,
    prompt: str,
    run: PESRun,
    attempt_no: int,
    *,
    on_turn: Callable[[PhaseTurn], None],
) -> tuple[str, dict, list[PhaseTurn]]:
    """调 LLMClient,翻译异常为 _SDKError(error_type=...)。"""
    from scrivai.exceptions import _SDKError
    from scrivai.pes.llm_client import _MaxTurnsError, _SDKExecutionError
    from claude_agent_sdk import (
        CLIConnectionError,
        ClaudeSDKError,
        ProcessError,
    )

    try:
        resp = await self._llm.execute_task(
            prompt=prompt,
            system_prompt=self.config.prompt_text + "\n\n" + phase_cfg.additional_system_prompt,
            allowed_tools=phase_cfg.allowed_tools,
            max_turns=phase_cfg.max_turns,
            permission_mode=phase_cfg.permission_mode,
            cwd=self.workspace.working_dir,
            on_turn=on_turn,
        )
        return resp.result, resp.usage, resp.turns
    except _MaxTurnsError as e:
        raise _SDKError("max_turns_exceeded", str(e)) from e
    except _SDKExecutionError as e:
        raise _SDKError("sdk_other", str(e)) from e
    except (CLIConnectionError, ProcessError, ClaudeSDKError, RuntimeError) as e:
        raise _SDKError("sdk_other", str(e)) from e
    # KeyboardInterrupt / CancelledError 不接,直接冒泡到 BasePES.run()
```

### 5.3 `_SDKError` 异常(在 `scrivai/exceptions.py` 加)

class _SDKError(Exception):

## 7. design.md 同步范围

### 7.1 §5.1.3 `_run_phase()` 伪代码

**Step 4 SDK 调用段** —— 替换:

| 原 | 改后 |
|---|---|
| `except RateLimitError as e: error_type="sdk_rate_limit"` | `except _SDKError as e: error_type=e.error_type` |
| `except MaxTurnsExceeded as e: error_type="max_turns_exceeded"` | (合并到 `_SDKError`,由 `LLMClient` 边界分类) |
| `except AnthropicError as e: error_type="sdk_other"` | (合并到 `_SDKError`) |

加 1 段说明:**SDK 0.1.61 不通过异常表达错误,而是通过 `ResultMessage.is_error` + `stop_reason` 字段。`LLMClient` 在边界把消息流错误翻译为 `_MaxTurnsError` / `_SDKExecutionError`,`BasePES._call_sdk_query` 再统一包成 `_SDKError(error_type=...)`,`_run_phase` 据此组装 PhaseError。**

### 7.2 §5.3.4 `is_retryable` 决策表

| `error_type` | 修改前 | 修改后 |
|---|---|---|
| `sdk_rate_limit` | `is_retryable=True` | **删除此条**(SDK 0.1.61 不区分,归并到 sdk_other) |
| `max_turns_exceeded` | `is_retryable=True` | 不变 |
| `sdk_other` | `is_retryable=False` | **改 `is_retryable=True`**(rate limit 现归这里,L2 必须能重试) |
| `response_parse_error` | `is_retryable=False` | 不变 |
| `output_validation_error` | `is_retryable=True` | 不变 |
| `cancelled` | `is_retryable=False` | 不变 |
| `hook_error` | `is_retryable=False` | 不变 |

### 7.3 §5.3.4 三层重试机制表

| 修改前 | 修改后 |
|---|---|
| L1 传输级(`_call_sdk_query` 内退避 3 次)/ L2 Phase 级 / L3 Run 级 | **L2 Phase 级 / L3 Run 级**;L1 标注:"M1.0 不实现,推迟到 M2 T2.5 视压测结果决定" |

### 7.4 §5.3.4 完整错误 → 状态映射表

删除两行:
- "SDK `RateLimitError` 首次 / L1 内自动退避"
- "SDK `RateLimitError` L1 耗尽 → `sdk_rate_limit`"

合并到一行:
- "**任意 SDK 失败 → `sdk_other`,`is_retryable=True`,L2 phase 级重试** —— 包括 rate limit / 网络超时 / SDK 内部错误。L2 重试耗尽 → `run.status="failed"`。"

### 7.5 §5.3.6 SDK 依赖暴露面段落

整段重写。原文说"`BasePES._run_phase` 直接 except RateLimitError / MaxTurnsExceeded / AnthropicError" —— 0.1.61 不存在这些异常。改为:

> SDK 0.1.61 通过**消息字段**(`ResultMessage.is_error` / `AssistantMessage.error`)而非异常表达错误。Scrivai 的策略:**`LLMClient`(`scrivai/pes/llm_client.py`)在边界把消息流错误翻译为内部异常**(`_MaxTurnsError` / `_SDKExecutionError`),`BasePES._call_sdk_query` 再映射为 `error_type` 字符串。这是单点适配,SDK 升级或更换供应商时,只需改 `LLMClient`,`BasePES` 零改动。
>
> 多供应商:`ModelConfig.base_url + api_key` 通过 `ClaudeAgentOptions(env={"ANTHROPIC_BASE_URL": ..., "ANTHROPIC_AUTH_TOKEN": ...})` 注入 SDK 子进程 env;主进程 env 不污染,并发安全。
>
> 未来方案(M2+)不变:引入 `AgentDriver Protocol` 把 `LLMClient` 抽象化,以支持非 SDK 供应商(如自研 Agent loop)。

### 7.6 §5.3.7(新增段)`LLMClient` 接口约定

新增 1 段:

> `LLMClient.execute_task(...)` 是 BasePES 与 SDK 之间的唯一适配点。接口约定:
> - 输入:`prompt / system_prompt / allowed_tools / max_turns / permission_mode / cwd / extra_env / on_turn`
> - 输出:`LLMResponse(result, turns, usage, duration_ms, session_id)`
> - 异常:`_MaxTurnsError` / `_SDKExecutionError` / `RuntimeError("未收到 ResultMessage")` / SDK 原生异常透传(`CLIConnectionError` / `ProcessError` / `ClaudeSDKError`)
> - 不做内部重试 —— L2 phase 级重试由 BasePES 负责
>
> 后续若多供应商需求来,只需写新 driver 实现同接口,`BasePES` 零改动。
